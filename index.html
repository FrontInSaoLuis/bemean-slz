Aprenda a criar uma aplicação completa usando somente Javascript

O workshop ensinará como criar sistemas inteiros apenas com Javascript, para isso utilizando no frontend o AngularJS que é um framework de Javascript mantido pelo Google, no back-end o tão falado Node.js que executa o Javascript no backend usando a engine V8 do Google e usando um banco de dados NoSQL com interface em Javascript, o igualmente famoso MongoDB. Além da teoria básica sobre os conceitos abordados, o workshop é puramente mão na massa, a didática é baseada em demonstrações de código com a explicação teórica em cima do código, criando um workflow para a criação de um sistema integrado entre todas as partes. O aluno terminará o workshop entendendo como cada parte se integra para criar uma aplicação REST

Como o curso é realizado?

Iniciaremos conhecendo o MongoDb e como utilizá-lo de diversas formas através do Javascript em seu console, após essa base de banco de dados não relacional conheceremos o Node.js, o qual foi o maior divisor de água no mundo do Javascript, após conseguirmos criar um CRUD utilizando Node.js e MongoDb (Mongoose), com rotas em Express avançaremos para o Frontend onde tudo muda e teremos que nos ambientar ao conceito de Single Page Application. Com isso faremos com que o AngularJs requisite os dados de nosso webserver via REST e monte a view corretamente, fazendo um integração completa usando apenas Javascript. Curso terá a duração de 20h.
